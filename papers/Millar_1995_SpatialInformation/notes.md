# Understanding and Representing Spatial Information

**Authors:** Susanna Millar
**Year:** 1995
**Venue:** British Journal of Visual Impairment, 13(1), 8–11
**DOI:** 10.1177/026461969501300102

## One-Sentence Summary
Millar explains how total absence of vision specifically disrupts spatial reference frame construction — not cognitive ability — and argues that assistive interventions must integrate substitute cues with existing body-centred coding rather than simply adding new modalities.

## Problem Addressed
Why do congenitally totally blind children struggle specifically with spatial tasks requiring mental reorganisation, despite demonstrating strong cognitive abilities in non-spatial domains (e.g. chess)?

## Key Contributions
- Identifies two specific informational deficits caused by total blindness: (1) loss of direct information about external surface relations, and (2) loss of reference cue redundancy from cross-modal overlap
- Reframes body-centred reference and movement coding as rational adaptations to informational conditions, not developmental delay
- Argues that tactile representations (maps, graphs, raised-line drawings) should be introduced through production (movement planning) before recognition, reversing the visual assumption
- Demonstrates that congenitally blind children can draw and represent 3D objects in 2D using the same schematic strategies as sighted children, with predictable differences explained by perceptual information availability

## Methodology
Review/commentary paper summarising findings from Millar's book (Millar, 1994) — a programme of empirical studies with congenitally totally blind and sighted children. No new data presented; this paper discusses practical implications.

## Implementation Details

### Spatial Reference Framework
The core theoretical model identifies two key factors for spatial coding:
1. **Specialisation**: Sensory systems analyse different complementary aspects of information from internal (body-centred) and external sources
2. **Redundancy**: Information from these sources typically converges and partially overlaps in normal conditions — this overlap is critical for development

### Why Blindness Disrupts Spatial Coding (Two Specific Effects)
1. Little direct/reliable information about relations between external surfaces → missing spatial frame cues
2. Loss of redundancy from cross-modal overlap (vision normally overlaps with vestibular, proprioceptive, kinaesthetic cues for gravitational vertical)

### Body-Centred Coding Is Rational, Not Primitive
- Blind children prefer body-centred reference frames and movement memory because these are more constant and reliable than external cues in the absence of vision
- This is NOT developmental delay — it requires MORE cognitive skill
- Using one hand as anchor for exploration by the other is a specific manifestation of this strategy

### Design Principles for Assistive Spatial Systems
1. **Substitute information must converge with existing reference cues** — adding a new modality alone is insufficient
2. **Integration must be explicit** — the link between substitute information and current coping strategies must be made obvious
3. **Sound substitution (including echo-location) is necessary but not sufficient** — it provides distance/direction but not surface-relation knowledge unless integrated with body-centred knowledge
4. **The blind infant needs to understand the analogy between body planes/surfaces and external planes/surfaces**
5. **Mobility training should integrate reference information from different sources** with the child's existing coding, not just emphasise distance cues

### Tactile Representation (Maps, Graphs, Drawings)
- **Production before recognition**: For touch (unlike vision), producing a configuration is EASIER than recognising one — systematic movements needed for recognition require prior knowledge of expected configurations
- **Movement planning as entry point**: Initial learning with graphic illustrations should be through movement planning (tracing raised lines) before attempting recognition
- **Symbol systems require explicit instruction**: The assumption that sighted people "just see" map/graph meanings is wrong even for sighted users; for blind users, the gap is larger but the same problem
- **Movement imagery as rehearsal mechanism**: Movements can be covertly rehearsed, providing a non-verbal mental representation analogous to subvocal speech rehearsal

### Drawing by Blind Children
- Children >10 years (blind and blindfolded sighted) use the same schematic strategies for representing 3D objects in 2D (circles for heads, lines for arms)
- Younger blind children use scribbles as tokens (sighted peers at same age use shapes) — explained by the difference in perceptual information, not cognitive deficit
- Blind children oriented figures horizontally on the page — understood the representation but lacked visual convention for depicting upright orientation in 2D

## Results Summary
This is a review/commentary paper, not an empirical study. Key claims summarised from the underlying research programme (Millar, 1994):
- Body-centred spatial coding in blind children is informationally rational, not a developmental deficit
- Cross-modal reference redundancy can be partially restored through coordinated, assisted learning
- Tactile representations work best when introduced through production (movement planning) first
- Many blind children lose interest in spatial problems due to insufficient integrated information — not inability
- Many electronic aids are rejected because the link to existing coping strategies is not made explicit

## Limitations
- Based on a single book's (Millar, 1994) research programme — inherits those studies' limitations
- No new empirical data presented
- Focuses on congenitally totally blind children — does not address late-onset blindness or partial vision
- Does not provide specific quantified recommendations (e.g., ages, training durations, measurable outcomes)
- 1995 publication — predates modern assistive technology (smartphones, GPS navigation, 3D audio)

## Testable Properties
- Body-centred reference coding should predominate over external reference coding in congenitally blind participants, compared to sighted controls, in spatial tasks
- Integration of substitute cues with existing body-centred coding should improve spatial task performance more than adding substitute cues alone
- Producing tactile configurations should be easier than recognising them for blind participants (reverse of the visual pattern)
- Blind children should use the same representational schemas as sighted children for drawing, with predictable developmental lag explained by perceptual information availability

## Relevance to Project
This paper provides the foundational cognitive framework for understanding WHY blind users interact with spatial information differently — not as a deficit but as a rational adaptation to informational conditions. This directly informs: (1) the design of accessible map representations (connecting to Biggs_2025_MapEquivalentPurpose's finding that text maps score poorly on spatial equivalence), (2) why simply adding audio or tactile overlays to spatial content fails without integration with existing user strategies, and (3) the theoretical basis for why "equivalent purpose" for spatial content requires understanding the user's reference frame system, not just presenting the same data in a different modality.
