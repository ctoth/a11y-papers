Westin (2004) documents Terraformers, a commercial real-time 3D action game developed over three years to be simultaneously playable by blind, low-vision, and sighted players, using a suite of spatial audio techniques (3D Sound Compass, Sonar, GPS) and voiced hierarchical menus as complete non-visual substitutes for the graphical interface. A post-mortem email survey of four blind paying customers found all accessibility features rated easy to use, with the main complaints being game crashes due to unstable GPU drivers and difficulty balance inconsistencies. This paper is foundational for game accessibility research: it catalysed the IGDA Game Accessibility Special Interest Group and provides the earliest detailed implementation-level case study of non-visual 3D game design using spatial audio.
