## Key Citations for Follow-up

- **Call-Jimenz and Lujan-Mora (2016)** - "Web Accessibility Barriers in Geographic Maps" - Directly quantifies web map accessibility barriers; the empirical complement to Hennig's recommendations. Relevant to web standards and evaluation methods areas.

- **Jenny and Kelso (2007)** - "Color Design for the Color Vision Impaired" in Cartographic Perspectives - Foundational work on color-safe map design, repeatedly cited for specific visual variable recommendations. Directly relevant to visual impairment research area.

- **Brock and Jouffrais (2015)** - "Interactive Audio-tactile Maps for Visually Impaired People" in ACM SIGACCESS - Represents the multimodal map approach (audio + tactile) that goes beyond Hennig's web-based focus. Relevant to assistive technology area.

- **Zeng and Weber (2010)** - "Audio-Haptic Browser for a Geographical Information System" - The Zeng and Weber series (2010, 2011, 2012a, 2012b) develops the audio-haptic approach to GIS access for VI users, a complementary modality to Hennig's visual + verbal approach.

- **Leporini and Paterno (2008)** - "Applying Web Usability Criteria for Vision-Impaired Users" - Empirically tests whether web usability criteria actually improve task performance for VI users, directly relevant to Hennig's usability argument.
