Singh et al. (2024) present FigurA11y, a human-AI collaborative system for writing scientific alt text that combines a training-free GPT-4 pipeline — conditioned on extracted figure metadata (caption, mentioning paragraphs, EasyOCR text, data tables) and DIAGRAM Center/SIGACCESS accessibility guidelines organized by figure type — with two interactive authoring features: Generate at Cursor (on-demand continuation/infilling at the author's cursor position) and Potential User Questions (auto-generated ambiguity questions with suggested answers). A within-subjects user study (N=14 paper authors describing their own figures) found that generated drafts enabled rapid description initiation and that Interactive Assistance prompted more iterative refinement without increasing cognitive load or manual effort relative to the base Draft+Revise version. For this collection, the paper provides the state-of-the-art implementation blueprint for AI-assisted scientific alt text authoring, including full prompt templates (Appendix B), the metadata extraction pipeline, and a validated design connecting to Wang_2021_ScientificPDFAccessibility's finding that alt text is the most urgent unaddressed gap in scientific document accessibility.
