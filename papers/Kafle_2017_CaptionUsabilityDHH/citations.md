## Key Citations for Follow-up

[25] **Kushalnagar, Lasecki, Bigham (2014) — "Accessibility evaluation of classroom captions"** (ACM TACCESS): The most cited prior work on DHH captioning accessibility. Evaluates real-time caption quality in classroom settings — essential context for the ACE paper and the natural next acquisition for this collection.

[22] **Kafle & Huenerfauth (2016) — "Effect of speech recognition errors in text understandability for DHH"** (SLPAT/Interspeech): The prior DHH study whose data was used to tune ACE's alpha parameter (0.65). Reading this would reveal exactly what comprehension tasks and stimuli were used as the calibration dataset.

[23] **Kawas, Karalis, Wen, Ladner (2016) — "Improving real-time captioning experiences for DHH students"** (ASSETS 2016): Real-time captioning system evaluated with DHH students — complements the ACE metric evaluation framework with a deployed system perspective.

[26] **Lasecki & Bigham (2014) — "Real-time captioning with the crowd"** (ACM interactions): Crowd-sourced captioning as a complement or alternative to ASR. Relevant because it provides the architectural context in which caption quality metrics like ACE would be applied.

[32] **Morris, Maier & Green (2004) — "From WER and RIL to MER and WIL"** (Interspeech 2004): The prior art on WER alternatives (Match Error Rate and Word Information Loss). ACE explicitly supersedes these for DHH captioning — useful for situating ACE in the broader ASR evaluation literature.
