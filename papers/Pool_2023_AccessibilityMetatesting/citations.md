## Key Citations for Follow-up

**[13] Silva et al. (2019)** — "Types of Problems Encountered by Automated Tool Accessibility Assessments, Expert Inspections and User Testing: A Systematic Literature Mapping." ACM ASSETS 2019. Directly relevant: the only citation in this paper that compares automated tools against human expert inspection and user testing — establishes which problem types each evaluation method can and cannot find. Critical complement to Pool's automated-only study.

**[8] Kumar, Venkatesh, and Biswas (2021)** — "Comparing Ten WCAG Tools for Accessibility Evaluation of Websites." Technology and Disability. Largest tool count (10) in the comparative literature; analyzes WCAG principle coverage (Perceivable, Operable, Understandable, Robust). Useful for cross-validating Pool's findings with a different corpus.

**[3] Burkard, Zimmermann, and Schwarzer (2021)** — "Monitoring Systems for Checking Websites on Accessibility." Frontiers in Computer Science. Evaluates commercial (paid) monitoring tools rather than free/open tools — complementary to Pool's free-tool focus; finding that fewer-issue tools still catch unique violations aligns with Pool's complementarity result.

**[1] Abduganiev (2017)** — "Towards Automated Web Accessibility Evaluation: A Comparative Study." Earliest large-scale comparative study in Pool's citation set (8 tools, 52 pages); establishes the prior that tools are complementary and that the landscape changes rapidly. Good historical baseline.

**[10] Pădure and Pribeanu (2020)** — "Comparing Six Free Accessibility Evaluation Tools." Government website corpus (Romanian municipalities); another independent replication of the complementarity finding with a different geographic/domain context.
